# Algebra 2 — Credit by Exam (UT High School)

UT High School CBE **study guides** for Algebra 2. UT splits each course exam into two
halves by semester; **both must be passed to earn the full year of credit**.

- [`uths-alg2-A.pdf`](./uths-alg2-A.pdf) — Exam **A** (first semester): format, TEKS topics, sample items, allowed materials.
- [`uths-alg2-B.pdf`](./uths-alg2-B.pdf) — Exam **B** (second semester).

**Prerequisite:** Algebra 1 only (Geometry is *not* required first).

**Calculator:** a graphing calculator is required; CAS models (TI-89, TI-Nspire CAS) are
prohibited — see [`../../../cbe/uths-calculator.pdf`](../../../cbe/uths-calculator.pdf).

**Rules & scoring:** see the main summary at [`/git/school/cbe/README.md`](../../../cbe/README.md).
Algebra 2 has **no STAAR EOC**, but it **is a prerequisite for Precalculus** — so failing
this exam blocks Precalculus-by-exam.
